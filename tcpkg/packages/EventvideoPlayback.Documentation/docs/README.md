# EventVideoPlayback.Documentation

Comprehensive documentation package for the EventVideoPlayback system including installation guides, usage instructions, system requirements, and service documentation.

## What's Included

- Installation and setup guides
- API documentation and reference
- Architecture diagrams
- Sample code and examples
- System requirements
- Service configuration documentation

## Installation

Install via TwinCAT Package Manager:
```
tcpkg install Beckhoff-USA-Community.XAE.Documentation.EventVideoPlayback
```

Documentation is deployed to the TwinCAT installation directory's Documentation folder.

## Online Access

Documentation is also available online at: https://github.com/Beckhoff-USA-Community/EventVideoPlayback
