# EventVideoPlayback.XAR

TwinCAT XAR workload package bundling the EventVideoPlayback system service for runtime environments.

## What's Included

This workload installs:
- EventVideoPlayback Windows Service
- OpenCV processing libraries
- Video codec dependencies
- Service configuration files

## Features

- Background video processing service
- Automatic image-to-video conversion
- H.264 video encoding
- Industrial PC optimized

## Installation

Install via TwinCAT Package Manager:
```
tcpkg install Beckhoff-USA-Community.XAR.EventVideoPlayback
```

The service will be automatically installed and started.

## Requirements

- Windows 10 IoT Enterprise or Windows 11
- .NET Framework 4.8 or higher
- TwinCAT 3 XAR Runtime
